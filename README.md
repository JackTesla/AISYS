# AISYS

项目介绍：用于AI生成图像的Web展示平台与后台管理平台。
JAVA作为后端编程语言，采用Spring+SpringMVC+Mybatis的框架，Mysql作为数据库，
基于web平台用AI生成图片并展示，同时web可以管理服务器，远程执行shell、查看服务器运行状态等。
